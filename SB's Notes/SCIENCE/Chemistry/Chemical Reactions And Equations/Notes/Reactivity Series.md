## Memory Trick
You can easily memorize the order of the core elements using this common mnemonic: [5, 12]

- Please (Potassium)
- Stop (Sodium)
- Calling (Calcium)
- Me (Magnesium)
- A (Aluminium)
- Careless (Carbon)
- Zebra (Zinc)
- Instead (Iron)
- Try (Tin / Lead)
- Learning (Lead)
- How (Hydrogen)
- Copper (Copper)
- Mercury (Mercury)
- Saves (Silver)
- Gold (Gold) [5, 13, 14, 15]
